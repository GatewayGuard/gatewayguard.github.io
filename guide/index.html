Security Settings Guide

Plain-English explanations for all 19 Windows 11 security settings GatewayGuard checks. Click any setting to see what it does, why it matters, and how to check or change it yourself.

Security — GatewayGuard handles these automatically

Settings GatewayGuard checks and fixes for you

For these eight settings, GatewayGuard checks your current status and makes the recommended change with your approval. You don’t need to do anything manually.

Setting 1

Windows Update

Makes sure your PC downloads security patches automatically. Outdated Windows is the single most common way PCs get compromised.

GatewayGuard checks this

Setting 2

Defender Real-Time Protection

Windows’s built-in antivirus. When it’s on, it catches threats as they happen instead of after the damage is done.

GatewayGuard checks this

Setting 3

Tamper Protection

Stops malware from turning off your antivirus. If this is off and you didn’t turn it off, that’s a red flag worth investigating.

GatewayGuard checks this

Setting 4

SmartScreen

Warns you before you open a file or visit a website that’s known to be dangerous. An extra safety net for accidental clicks.

GatewayGuard checks this

Setting 5

Defender Periodic Scanning

Lets Windows Defender run a second opinion scan even if you have another antivirus installed. Catches things that slip through the gaps.

GatewayGuard checks this

Setting 6

Enhanced Phishing Protection

Warns you when you type your Windows password into a website or unsecured document. Helps stop password theft before it happens.

GatewayGuard checks this

Setting 7

Windows Firewall

Blocks uninvited connections trying to reach your PC over the network. Should be on for all three network profiles: Private, Public, and Domain.

GatewayGuard checks this

Setting 8

BitLocker Drive Encryption

Scrambles everything on your hard drive so it’s unreadable without your password. Protects your files if your laptop is ever lost or stolen.

GatewayGuard checks this

Manual setup required

Settings you need to set up yourself

These settings require steps Windows won’t let a script handle automatically. GatewayGuard walks you through each one and directs you here for the step-by-step instructions.

Setting 9

Windows Hello

Sign into your PC with your face, fingerprint, or a PIN instead of typing a long password every time. More secure and more convenient.

Manual setup needed

Setting 10

Remote Desktop

Lets someone control your PC over the internet. Most home users should have this turned off. Windows Pro only.

Manual setup needed

Setting 16

Memory Integrity

Protects the core of Windows from attacks that try to sneak in through device drivers. Requires a reboot to take effect.

Manual setup needed

Setting 17

Password Required on Wake

Makes your PC ask for your password when it wakes from sleep. Stops anyone from sitting down at your unlocked PC while you step away.

Manual setup needed

Convenience settings — you decide

Features you can turn off to improve privacy

These settings are harmless to turn off — they’re convenience features, not security ones. GatewayGuard explains each and lets you decide. There’s no wrong answer.

Setting 11

Advertising ID

A unique number Windows assigns to your account so advertisers can track which ads you see. Turning it off has no effect on how Windows works.

Your choice

Setting 12

Diagnostic Data

Information Windows sends to Microsoft about how you use your PC. You can set this to “Required only” instead of sending everything.

Your choice

Setting 13

Edge Startup Boost

Keeps parts of the Edge browser running in the background so it opens faster. Turning it off frees up memory and stops Edge from running when you didn’t ask it to.

Your choice

Setting 14

Windows Widgets

The news and weather panel on your taskbar. It sends usage data back to Microsoft. Turning it off removes the panel and stops the data sharing.

Your choice

Setting 15

Edge Password Saving

Edge’s built-in offer to save and fill your passwords. Convenient, but most security experts recommend a dedicated password manager instead.

Your choice

System hardening

Settings that close less obvious security gaps

These two settings are less well-known but worth addressing. GatewayGuard explains the trade-offs and asks before making any change.

Setting 18

Fast Startup

Makes Windows start up faster by saving part of its state to disk. The trade-off: your PC isn’t fully shut down when you close the lid, which can interfere with updates and encryption.

GatewayGuard checks this

Setting 19

Wake on LAN

Lets your PC be turned on remotely over the network. Useful in offices, unnecessary at home. GatewayGuard checks whether it’s enabled and explains the risk.

GatewayGuard checks this
